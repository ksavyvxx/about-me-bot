# 🚀 Telegram-Бот Портфолио | Конкурс «Обо мне» (CapEducation)

Привет! Это мой проект для конкурса от CapEducation. Я создал интерактивного Telegram-бота, который не просто рассказывает обо мне и моих успехах, но и использует современные инструменты разработки такие как интеграция нейросети Gemini и многое другое.

---

## ✨ Ключевые фичи проекта

*   Спроси ИИ обо мне: Бот подключен к API `gemini-2.5-flash`. Нейросеть знает контекст (мою биографию, цели, инфо о менторе) и отвечает на любые свободные вопросы пользователя строго по фактам из моей жизни.
*   Данные отдельно от кода: Все тексты, описания проектов и ссылки лежат в одном файле `content.json`. Если я захочу изменить описание хобби, мне не придется править код бота.
*   Умный запуск: Токены и ключи можно передавать прямо при старте через аргументы командной строки или переменные окружения.
*   Валидация отзывов: Раздел обратной связи защищен регулярными выражениями. Бот проверяет правильность ввода формата `email: текст` и не пропускает спам.
*   Удобный интерфейс: Навигация работает тремя путями — через текстовые команды (`/start`, `/about`), постоянное кнопочное меню (Reply) и динамические кнопки под сообщениями (Inline).
*   Пасхалка:В коде зашито секретное слово. Попробуй угадать его или найти в коде `validators.py`, чтобы получить секретную пасхалку!

## Структура файлов проекта

```text
about_me_project/

* config.py          # Чтение аргументов запуска, проверка переменных и загрузка JSON
* content.json       # Текстовая база данных (информация обо мне и проектах)
* validators.py      # Проверка формата email в отзывах и поиск пасхалок (модуль re)
* main.py            # Главный файл: логика кнопок, команд, Middleware-логи и ИИ
* requirements.txt   # Зависимости (pyTelegramBotAPI, google-genai)          
* README.md          # Документация, которую вы сейчас читаете

---

## Чтобы запустить бота, введите в терминал следующие команды:

# Установка необходимых зависимостей
pip install -r requirements.txt

# Запуск бота с передачей токенов (замените на свои значения)
python main.py --token=ваш_токен_телеграм_бота --gemini=ваш_ключ_gemini
